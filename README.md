# 基于 AI 的大学生职业规划智能体

本项目是一个可运行的职业规划智能体原型，覆盖以下能力：

- 岗位数据解析与岗位画像构建（支持 10000 条级别文件）
- 岗位关联图谱（垂直晋升路径 + 换岗路径）
- 学生就业能力画像（简历上传/文本录入）
- 人岗匹配评估（基础要求、职业技能、职业素养、发展潜力四维）
- 职业生涯发展报告生成（可编辑、完整性检查、润色、导出）

## 1. 快速启动

1. 安装依赖

   ```powershell
   pip install -r requirements.txt
   ```

2. 配置大模型（可选但推荐）

   复制 `.env.example` 为 `.env`，填写以下变量：

   - `LLM_API_KEY`
   - `LLM_BASE_URL`
   - `LLM_MODEL`

   未配置时会使用规则引擎回退，依然可运行。

3. 启动应用（Streamlit 原型）

   ```powershell
   streamlit run app.py
   ```

4. 启动应用（网页版 FastAPI）

   ```powershell
   uvicorn web_app.main:app --reload --host 0.0.0.0 --port 8000
   ```

   本机浏览器访问：`http://127.0.0.1:8000`

   其他电脑访问：`http://<本机局域网IP>:8000`

   如果要让别的电脑访问，运行时必须监听 `0.0.0.0`，并放行 Windows 防火墙的 `8000` 端口。

## 6. 如果其他电脑还是访问不了

请按下面顺序排查：

1. 确认访问地址是 `http://192.168.1.100:8000`，不要用 `127.0.0.1`。
2. 确认另一台电脑和这台电脑在同一个局域网、同一个网段，优先检查是否也是 `192.168.1.x`。
3. 确认 Windows 防火墙允许 `8000` 端口的入站连接。
4. 确认没有使用访客 Wi-Fi、隔离网络或 VPN，这些网络可能禁止设备互访。
5. 先在这台电脑上测试 `http://127.0.0.1:8000` 和 `http://192.168.1.100:8000`，如果这两个都正常，问题通常就在网络或防火墙，而不是程序本身。

如果需要手动放行端口，可以用管理员权限运行：

```powershell
netsh advfirewall firewall add rule name="CareerAgent 8000" dir=in action=allow protocol=TCP localport=8000
```

5. Windows 一键启动（推荐）

   - 双击 `start_web.bat`：自动检测 Python、安装缺失依赖、打开浏览器并启动服务，且默认监听所有网卡，便于局域网其他电脑访问。
   - 双击 `start_web.vbs`：静默方式启动（不弹出命令窗口）。
   - 双击 `stop_web.bat`：停止 8000 端口上的服务进程。

## 2. 数据格式

请上传文件（支持 .csv/.xls/.xlsx），字段建议包含：

- 职位名称
- 工作地址
- 薪资范围
- 公司全称
- 所属行业
- 人员规模
- 企业性质
- 职位编码
- 职位描述
- 公司简介

项目已提供示例数据：`data/sample_jobs.csv`。

## 3. 技术实现说明

- `app.py`: Streamlit 交互原型
- `web_app/main.py`: Web API 与页面路由
- `web_app/templates/index.html`: 网页前端结构
- `web_app/static/*`: 网页样式与脚本

### 3.1 岗位画像（不少于 10 个）

- 对岗位描述进行技能、证书、能力标签抽取。
- 按岗位名称聚合生成画像。
- 自动筛选样本量较高岗位，默认输出 Top 10+ 岗位画像。

### 3.2 岗位图谱

- 垂直岗位图谱：根据岗位级别词（初级/中级/高级/专家/总监）和相似岗位族构建晋升链。
- 换岗路径图谱：基于技能重叠度与岗位族相关性生成血缘换岗路径。
- 至少提供 5 个岗位，每个岗位不少于 2 条换岗路径。

### 3.3 学生能力画像

- 输入来源：简历上传（TXT/PDF/DOCX）或手动录入。
- 输出维度：专业技能、证书、创新能力、学习能力、抗压能力、沟通能力、实习能力等。
- 给出完整度评分与竞争力评分。

### 3.4 人岗匹配评分

评分维度（可调权重）：

- 基础要求
- 职业技能
- 职业素养
- 发展潜力

综合得分公式：

$$
MatchScore = \sum_{i=1}^{4} w_i \times s_i
$$

其中，$\sum w_i = 1$，每个子项 $s_i \in [0, 100]$。

### 3.5 报告生成与导出

- 生成人岗匹配分析、职业目标设定、路径规划、行动计划。
- 支持报告完整性检查与润色优化。
- 支持一键导出 Markdown。

## 4. 准确性评估建议

项目包含抽样评估入口，可对以下指标进行监测：

- 关键技能匹配准确率（目标 >= 80%）
- 岗位/学生画像关键信息准确率（目标 >= 90%）

建议结合标注样本在 `evaluation` 模块持续评估。
